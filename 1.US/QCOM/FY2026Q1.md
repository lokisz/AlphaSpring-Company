## 一、财报简评

这季整体是**“业绩 beat、指引 miss”**的典型组合：FY26 Q1（截至 2025-12-28）交出**营收 $12.252B（+5% YoY）/ Non-GAAP EPS $3.50（+3% YoY）**的 record quarter，核心由**旗舰安卓机 + 汽车**带动；但 FY26 Q2 guidance 因“**memory supply constraints（DRAM 供给紧 + 价格上行）**”明显低于市场预期，直接触发盘后股价大跌（Reuters 口径约 -9%）。

市场怎么看：**基本面没坏（需求端 OK），但短期“能不能出货”被 memory 卡脖子**，所以交易的是近 1–2 个季度的预期下修，而不是长期 story。管理层在电话会里也把话说得很直——“**It’s 100% related to memory**”。

## 二、公司信息

Qualcomm 是全球领先的**无线通信/移动 SoC + 连接 + 端侧 AI**的 fabless 芯片公司，商业逻辑是“**QCT 卖芯片 + QTL 收专利费**”，在安卓高端 SoC/基带上长期强势；主要竞争/替代压力来自 **MediaTek（安卓 SoC）、Apple/Samsung 自研（垂直整合）、以及多家车载/工业 SoC 厂商**。

**业务拆分（FY26 Q1）：**

- **QCT（芯片业务）**：$10.613B（约占总营收 **86.6%**）
    
    - Handsets：$7.824B（占 QCT **73.7%**）
        
    - Automotive：$1.101B（占 QCT **10.4%**）
        
    - IoT：$1.688B（占 QCT **15.9%**）
        
- **QTL（授权）**：$1.592B（约占总营收 **13.0%**）
    

> 备注：QCT+QTL 与总营收存在小额抵消/其他项，属正常口径差异。

**战略重点（本季强调项）：**

- **“memory shortage → handset 出货天花板”**：短期主要风险来自 DRAM 供给向 HBM/AI 数据中心倾斜，影响手机 OEM 生产计划与库存策略。
    
- **多元化（Auto/IoT/Data Center/Robotics）**：管理层持续强调 FY2029 diversification 目标，并提到数据中心 AI 芯片路线与收购（如 Alphawave Semi、Ventana）强化平台能力。

## 五、电话会要点

### 5.1 管理层核心观点（提炼）

- **Q1 需求端强**：旗舰机卖得不错，premium/high-tier sell-through 超预期。
    
- **Q2 的“坑”来自 memory**：不是 Qualcomm 需求差，而是 OEM 拿不到（或拿不起）足够 DRAM。
    
- **汽车与 IoT 继续加速**：汽车 Q2 YoY 增速指引 **>35%**（管理层口径）。
    
- **数据中心/机器人是中长期增量**：强调 inference、功耗/TCO 优势，并提到收购加强数据中心平台。
    

### 5.2 分析师问答（问题 → 回答要点 + 摘录）

1. **TD Cowen（Joshua Buchalter）**：handset weakness 除 memory 外还有别的因素吗？今年 TAM 怎么看？
    
    - 答：管理层明确表示“**100% related to memory**”，并指出宏观与 sell-through 并不弱。
        
2. **同一分析师追问**：汽车指引为何 QoQ 加速这么猛？
    
    - 答：管理层强调汽车收入主要来自**份额提升 + 新车项目爬坡**，“pipeline translate into revenues”。
        
3. **JPM（Samik Chatterjee）**：OEM 会不会在高端里“downshift”芯片档位来控成本？
    
    - 答：CEO 强调 premium tier 长期扩张趋势，且 OEM 会优先把资源给更赚钱的高端段位。
        
4. **Deutsche Bank（Ross Seymore）**：中国 exposure 多大？季节性怎么看？
    
    - 答：公司不按地区披露，但强调**按“tier 调整后”对中国暴露低于按 unit 直观推算**；需求季节性没变，关键在供给能否匹配需求。
        
5. **Bernstein（Stacy Rasgon）**：March handset $6B 后，June 怎么 model？QTL run-rate 呢？
    
    - 答：公司不对 Q2 以后做指引，但给了框架：**March 可作为 June 的建模参考**；QTL 也“pretty consistent with trend”，但仍受供给影响。
        
6. **UBS（Tim Arcuri）**：QCT op margin drop-through 看起来 >100%，是不是还有别的事？+ Huawei license 更新？
    
    - 答：公司称 gross profit 大体稳定，主要是收入 scale + OpEx；Huawei 许可谈判“**no update**”，细节不便披露。
        
7. **Cantor（C.J. Muse）**：若 shortage 拉很久如何规划？会不会导入国产内存（如 CXMT）？
    
    - 答：公司澄清“**For handsets, we don’t buy memory**”，并表示在多家内存供应商上都有认证与兼容弹性。

## 六、增长与利润率

**增长驱动力：**

- QCT 这季的“主引擎”仍是 **旗舰 handset**（YoY +3%）+ **汽车（YoY +15%）**；IoT 也回到正增长（YoY +9%）。
    
- 但 Q2 预期被 memory 卡住：管理层口径是“需求没问题”，供应链/价格把行业规模压住——有点像**“外面风平浪静，但港口限流”**。
    

**毛利率/利润率：**

- GAAP GM 约 **54.6%**（YoY、QoQ 均小幅下滑）；QCT EBT margin **~31%**，QTL EBT margin **~77%**（许可依旧是利润压舱石）。
    
- 可持续性判断：短期 margin 主要取决于 **handset 出货量（规模效应）**与 **高端产品 mix**。如果 memory 约束持续，margin 更像“**被迫在低挡位巡航**”；一旦供给松动，回到“正常转速”并不难（管理层也暗示 normalize 后会回归 prior run-rate）。

## 七、持续关注的老问题

1. **对手机周期依赖**：QCT 里 handset 仍占比高（~74%），一旦行业供给/需求扰动，盈利弹性会被放大。
    
2. **客户垂直整合（Apple/Samsung）+ MediaTek 竞争**：Reuters 明确提到 Apple/Samsung 自研与 MediaTek 竞争加剧是市场长期担心点之一。
    
3. **中国相关风险**：公司自己在 forward-looking risk factors 中也点名“业务集中在中国”的风险（本季又叠加中国 OEM 降库存）。
    
4. **Licensing 摩擦（如 Huawei license）**：电话会仍是 “no update”，属于悬而未决的 headline risk。
    
5. **库存/现金流**：Inventories 较上季末小幅上行，但 OCF 依然强（$4.965B）。

## 九、投资启示

### 9.1 投资主张（Thesis）

当下 QCOM 的位置更像：**基本面 OK（高端份额 + Auto/IoT 继续爬坡），但短期情绪被 guidance 下修带节奏**。分歧点在于：

- 多头看 **diversification（Auto/IoT/Data Center）+ 估值便宜 + 回购**；
    
- 空头盯 **handset 依赖 + memory 约束持续时间 + 客户自研/竞争**。  
    核心驱动组合：**利润率（规模恢复） + 交易流（情绪修复） + 估值再评级**。
    

### 9.2 关键假设（Key Assumptions）

1. **memory 约束在未来 2–4 个季度边际缓解**｜指标：Q2→Q3 handset 指引回升/渠道库存改善｜来源：公司 earnings call/指引｜频率：每季
    
2. **Auto 维持 >30% YoY 增速段**｜指标：QCT Automotive revenue（>35% YoY 指引兑现）｜来源：公司指引/财报｜频率：每季
    
3. **QTL 稳定（不出现大额 license 风险事件）**｜指标：QTL revenue/margin + Huawei 许可进展｜来源：电话会/公告｜频率：不定期
    
4. **股东回报持续**｜指标：回购金额/股数、FCF｜来源：现金流量表/季度回购披露｜频率：每季
    

### 9.3 催化剂时间轴（Catalysts）

**1–6 个月：**

- FY26 Q2 业绩与指引（验证 memory 影响是否扩大/收敛）
    
- 汽车收入加速兑现（>35% YoY 指引落地）
    
- Huawei license 任何“更新/落地/延迟”都可能成为股价催化或风险点
    

**6–18 个月：**

- 数据中心 AI 芯片路线推进：公司提到 2026 年下半年推进、FY27 贡献更明显（Reuters 口径）
    
- 端侧 AI 设备形态扩散（AI glasses/robotics 生态进展）
    

### 9.4 估值（Valuation）

- 当前股价约 **$148.89**（2026-02-05 报价时间见图）
    
- Yahoo 口径 **Forward P/E ~12.15**（用于衡量市场对未来盈利的折价）
    
- Macrotrends 给出的 P/E（口径可能与 Yahoo 不同）显示 QCOM 在历史区间内偏中低位附近徘徊。  
    **方法选择：**对 QCOM 这种成熟且强回购的公司，用 **Forward P/E（配合一致预期 EPS）**更贴近市场定价逻辑。
    

### 9.5 回报结构（Payoff）

- 市场一致预期 EPS（FY2026）：**$12.03**（MarketWatch 口径）
    

|情景|估值方法|假设倍数（Forward P/E）|目标价区间|相对现价隐含回报|
|---|---|--:|--:|--:|
|Bear|P/E × FY26E EPS|10×|~$120|**-19.2%**|

> 解读：当前价格隐含的是“**不确定但不崩**”的预期；若 memory 约束确认是阶段性扰动、且 Auto/IoT 抵消手机波动，估值回到 13–16×区间并不激进。

### 9.6 交易流（Flow）

短期钟摆明显偏情绪端：这次下跌更像 **guidance-driven sell-off**（after-hours 大跌），而不是 Q1 业绩本身差。  
（指数纳入/剔除、解禁：QCOM 为成熟公司，通常不是主要矛盾，本次未见官方披露的结构性 flow 事件。）

### 9.7 多空观点（Long Vs Short）

**多头清单：**

- 高端安卓旗舰需求韧性 + 份额稳（管理层提到三星 premium 份额预期）
    
- Auto 连续 record，且指引加速（>35% YoY）
    
- 现金流强 + 回购力度大（本季回购 $2.6B）
    
- 估值偏便宜（forward multiple 低位）
    

**空头清单：**

- memory 约束持续时间不确定（可能拖多个季度），直接压 handset 出货与利润率
    
- 客户自研（Apple/Samsung）+ MediaTek 竞争导致中长期份额/ASP 压力
    
- Licensing（Huawei）谈判不确定性仍在
    

### 9.8 结论（Conclusion）

三个决定性变量：

1. **memory 供给与价格**何时边际缓解（决定 handset 规模与短期预期修复）；
    
2. **Auto/IoT 的增长斜率**能否持续对冲手机波动；
    
3. **许可业务/大客户自研**相关风险是否出现新催化。
    

一句话总结：**基本面在、估值不贵，但情绪被 memory 把方向盘抢走了——短期“风大”，中期看能否回到“正常航速”。**
