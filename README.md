# AlphaSpring-Company 知识库


这是一个基于AI大语言模型构建的上市公司资料库，旨在利用AI自动整理全球上市公司的基本资料和最新业绩. 目前以美股上市公司为主, 目前包括约400家上市公司. 公司资料由AlphaSpring系列Agent生成, 也可在chatgpt.com搜索alphaspring获取GPT使用入口, 输入ticker生成系列报告.

---

## 📂 目录结构

* **[📁 美股市场日报](./0.DailyReport)** 每日市场总结, 以SP500表现为主, 受限于数据源,每日异动清单仅能获取部分
* **[📁 美股上市公司](./1.US)** 美股上市公司,文件夹名为ticker, 子目录下包括公司总览和季度业绩等多个文件
* **[📁 中国大陆上市公司](./2.China/)** A股上市公司,采用总览和业绩分析合一的单一文件格式,大部分缺少公开电话会信息, 部分以股权激励代替业绩指引
* **[📁 中国香港上市公司](./3.HK)** H股上市公司,采用总览和业绩分析合一的单一文件格式,大部分缺少公开电话会信息
* **[📁 日本上市公司](./4.Japan)** 日本上市公司,采用总览和业绩分析合一的单一文件格式
* **[📁 新加坡上市公司](./5.SGX)** 新加坡上市公司,采用总览和业绩分析合一的单一文件格式
* **[📁 中国台湾上市公司](./6.TW)** TW上市公司,采用总览和业绩分析合一的单一文件格式
* **[📁 韩国上市公司](./7.KR)** 韩国上市公司,采用总览和业绩分析合一的单一文件格式
* **[📁 欧洲上市公司](./8.EU)** 欧股上市公司,采用总览和业绩分析合一的单一文件格式
* **[📁 其他公司](./8.Others)** 非上市公司和其他地区公司
* **[📁 行业研究](./9.Industry/)** 简析特定子行业的价值主张,上下游和关键要素
* **[📁 专题研究](./9.Research)** 特定问题研究
