- **Reuters**：$0.853B（约 **11.4%**）
- **Global Print**：$0.490B（约 **6.6%**）([PR Newswire](https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html "https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html"))

## 九、投资启示

### 9.1 投资主张（Thesis）

当前 TRI 的矛盾点很清晰：**基本面仍在走“稳健增长 + margin 扩张”**，但市场情绪被 AI disrupt 摇摆，估值从“visibility premium”向“AI 风险折价”回摆。驱动组合更偏向：**【增长 + 估值再评级/再定价 + 交易流】**。([Reuters](https://www.reuters.com/business/thomson-reuters-reports-higher-fourth-quarter-revenue-2026-02-05/ "https://www.reuters.com/business/thomson-reuters-reports-higher-fourth-quarter-revenue-2026-02-05/"))

### 9.2 关键假设（Key Assumptions）

1. **Big 3 有机增长维持 ~9%** → 观察：分部有机增速/ACV → 来源：季报/电话会 → 频率：每季。([PR Newswire](https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html "https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html"))
    
2. **AI 产品不被快速 commoditize** → 观察：CoCounsel/Westlaw 相关增长、客户续费/净留存 → 来源：电话会/产品更新 → 每季。([The Motley Fool](https://www.fool.com/earnings/call-transcripts/2026/02/05/thomson-reuters-tri-q4-2025-earnings-transcript/ "https://www.fool.com/earnings/call-transcripts/2026/02/05/thomson-reuters-tri-q4-2025-earnings-transcript/"))
    
3. **Adj. EBITDA margin 每年 +100bp 路径可兑现** → 观察：公司给的 2026–2028 框架与当季 margin → 来源：指引/财报 → 每季。([PR Newswire](https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html "https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html"))
    
4. **FCF 2026 ~ $2.1B** → 观察：OCF、capex、FCF → 来源：财报 → 每季。([PR Newswire](https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html "https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html"))

### 9.8 结论（Conclusion）

三项决定性变量：

1. **Big 3 的有机增长能否守住高个位数**；
    
2. **AI 产品是否能持续证明“专业级差异化”并守住定价权**；
    
3. **margin/FCF 指引兑现度（+100bp / FCF ~ $2.1B）**。([PR Newswire](https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html "https://www.prnewswire.com/news-releases/thomson-reuters-reports-fourth-quarter-and-full-year-2025-results-302680103.html"))
    

一句话总结：**基本面仍在推进，估值在“AI 焦虑”里被压着，情绪钟摆偏冷——但公司用增长+现金流在争取时间窗口。**
