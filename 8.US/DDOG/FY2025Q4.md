## 二、公司信息

Datadog 是 **SaaS 形态的 observability + cloud security 平台**，以“统一采集/分析 metrics、logs、traces + 安全信号”切入，核心商业逻辑是 **land-and-expand（用量/模块扩张）**，竞争对手包括 Dynatrace、Elastic（Observability）、（并入 Cisco 的）Splunk、以及在安全侧与 PANW 等厂商的交叉竞争。([Stock Titan](https://www.stocktitan.net/sec-filings/DDOG/8-k-datadog-inc-reports-material-event-f8f9b4a8adcf.html "https://www.stocktitan.net/sec-filings/DDOG/8-k-datadog-inc-reports-material-event-f8f9b4a8adcf.html"))

**业务/产品拆分（公司不按产品披露收入占比，更多用 ARR/渗透率描述）**：

- **三大核心可观测性支柱**：Infrastructure Monitoring、Log Management、APM+DEM（均已做到 **$1B+ ARR**，其中 Infrastructure Monitoring **$1.6B+ ARR**；Log Management **$1B+ ARR**；FlexLogs 逼近 **$100M ARR**）。([The Motley Fool](https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/ "https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/"))
    
- **平台化渗透**：Q4 末 **84%** 客户使用 ≥2 个产品；**55%** 使用 ≥4 个产品；**33%** 使用 ≥6 个产品；**9%** 使用 ≥10 个产品（同比继续抬升）。([The Motley Fool](https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/ "https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/"))
    
- **客户结构（用 ARR cohort 看）**：约 **32,700** 客户；其中 **4,310** 家为 $100k+ ARR 客户（约占客户数 **13.2%**），贡献约 **90% ARR**；**603** 家为 $1M+ ARR 客户。([The Motley Fool](https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/ "https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/"))
    
- **地域结构（FY2024 口径，来自 10-K）**：North America **$1.874B（~69.8%）**，International **$0.810B（~30.2%）**。FY2025 未在本次新闻稿中给到同口径拆分。([美国证券交易委员会](https://www.sec.gov/Archives/edgar/data/1561550/000156155025000110/dd_annualxreportx2024.pdf "https://www.sec.gov/Archives/edgar/data/1561550/000156155025000110/dd_annualxreportx2024.pdf"))
    

**战略重点（AI 相关）**：

- 公司把 AI 分成两条：**AI for Datadog**（用 AI 提升平台自动化/效率，如 Bits AI SRE agent、MCP server 等）与 **Datadog for AI**（LLM Observability、AI Security 等）。([The Motley Fool](https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/ "https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/"))

## 五、电话会要点

### 5.1 管理层核心观点（提炼）

- **需求面**：管理层称“需求环境 broad-based 正向”，且非 AI 客群出现加速迹象。([The Motley Fool](https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/ "https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/"))
    
- **AI 与平台形态**：未来 observability 会走向“更多自动化 + agent”，Datadog 一边做 UI 给人用，一边把能力开放给 agent（MCP server）。([The Motley Fool](https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/ "https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/"))
    
- **竞争观**：竞争格局短期变化不大，管理层强调“observability 是 full-time job”，并认为 LLM observability 不应割裂于全栈可观测性。([The Motley Fool](https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/ "https://www.fool.com/earnings/call-transcripts/2026/02/10/datadog-ddog-q4-2025-earnings-call-transcript/"))
